# Django란?
Django는 장고라고 불리고 파이썬을 이용한 웹 개발을 도와주는 프레임워크다.

# 간단한 웹 개발 순서
1. Django 설치
    - 터미널 명령어 : pip install django (혹은 아나콘다 사용 시 conda install django)
1. 프로젝트 만들기
    - 터미널 명령어 : django-admin startproject 프로젝트명
1. 로컬서버 실행하기 : python manage.py runserver
